# ChaFee-FullStack-Cafe
A responsive full-stack coffee shop web application built with HTML, Tailwind CSS, JavaScript (Fetch API), and PHP/MySQL. Features a dynamic cart with real-time add/remove operations, seamless background database connection, and an interactive live order tracking simulation.

##Features
* **Responsive Modern UI:** Built with Tailwind CSS, featuring smooth hover effects and customized modal popups.
* **Dynamic Cart Management (CRUD):** Asynchronous item addition and deletion using JavaScript Fetch API without page reloads.
* **Backend Database Integration:** PHP scripts handling secure data transport to a MySQL database.
* **Live Order Tracking:** Interactive tracking simulation detailing order progress steps.

## Tech Stack
* **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6, Fetch API), jQuery
* **Backend:** PHP
* **Database:** MySQL

##  Local Setup Instructions
1. Clone or download this repository.
2. Move the project folder into your XAMPP `htdocs` directory.
3. Open `phpMyAdmin` and create a new database named `chafee`.
4. Import the provided SQL dump file (e.g., `chafee.sql`) into the database.
5. Start **Apache** and **MySQL** from your XAMPP Control Panel.
6. Run `http://localhost/Chafee` in your browser.
